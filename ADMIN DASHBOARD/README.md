# LeadHouse Admin Dashboard

Internal management dashboard for the LeadHouse mentorship platform.

## Setup

```bash
npm install
npm run dev    # http://localhost:8080
```

## Login
- Email: `admin@leadhouse.co.ke`
- Password: `Leadhouse@2026!`

## Features
- User & mentor management
- Mentor matching oversight
- Sessions, goals, journal moderation
- MCQ challenge builder
- Community & reports moderation
- Messages (view all platform conversations)
- Global search across all platform data
- Platform settings

See the root `README.md` for full documentation.
